## Goal

Establish the TypeScript React/Vite foundation for the simulator.

## Deliverables

- React + TypeScript + Vite app scaffold.
- TypeScript config for app, tooling, and tests.
- Verified package scripts for dev, typecheck, lint, test, build, and e2e.
- No Python backend and no JavaScript source files for app logic.

## Acceptance Criteria

- `pnpm typecheck` passes.
- `pnpm lint` passes.
- `pnpm test` passes.
- `pnpm build` passes.
- `pnpm test:e2e` runs a browser smoke test against the built app.
