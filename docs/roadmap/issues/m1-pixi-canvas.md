## Goal

Render the gear system as an interactive PixiJS canvas.

## Deliverables

- PixiJS canvas component with 60-node radial grid.
- Gear rendering with visible teeth, color, labels, and selection ring.
- Connection rendering for mesh and compound links.
- Select-and-drag interaction for gear placement.
- Playback-driven angle updates from the simulation frames.

## Acceptance Criteria

- The starter gear train renders on first load.
- Selecting a gear updates the inspector.
- Dragging a gear updates its stored position.
- Play/pause animates gear angle state from the deterministic solver.
