## Goal

Turn the simulator into a polished portfolio project.

## Deliverables

- Keyboard-accessible controls for core workflows.
- Responsive layout pass for laptop and tablet widths.
- Bundle-size/code-splitting review for PixiJS/Recharts chunks.
- README screenshots or GIFs.
- Clear portfolio framing around TypeScript, simulation math, canvas rendering, and Supabase.

## Acceptance Criteria

- No clipped or overlapping text in primary desktop/tablet views.
- Keyboard users can operate playback and inspector controls.
- README explains why the project matters and how it was engineered.
- Remaining speculative inspiration is clearly separated from grounded implementation.
