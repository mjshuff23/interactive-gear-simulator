## Goal

Provide starter systems that teach base-60 concepts without making the user start from a blank canvas.

## Deliverables

- Clock train example.
- 60/30/20/15 harmonic division example.
- Compound axle example.
- Base-60 fraction and angle example.
- Short in-app labels that identify the mathematical relationship.

## Acceptance Criteria

- Users can switch examples without losing the app shell.
- Each example has deterministic gear data and passes schema validation.
- Examples are small enough to inspect and edit manually.
