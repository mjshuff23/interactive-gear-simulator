## Goal

Add learning-oriented visualizations that help users reason in base 60.

## Deliverables

- Live ratio tracker for selected gear versus driver gear.
- Waveform chart derived from selected gear angle.
- Sexagesimal angle readout.
- Connection list or gear-train graph for the current system.

## Acceptance Criteria

- Visualization values update while playback is running.
- Readouts use base-60 language where appropriate.
- UI avoids unsupported scientific claims and frames the feature as math education.
